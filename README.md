Расширение для отображения статистики каналов на twitch

Статистика с сайтов:
- **Twitch Tracker**
- **Sullygnome**

![Screenshot](https://github.com/BeNDYGo/TwitchExtension/blob/main/preview/Screenshot.png)

## Работа:

1. Откройте стрим
2. Под никнеймом стримера добавится окно
3. Выберите статистику с нужного сайта
4. Откроется статистика стримера

## Описание:

Расширение добавлят под стрим 2 кнопки для просмотра статистики открытого канала на *Twitch Tracker* и *Sullygnome*

## Особенности:
Для удобоства, при выборе сайта *Twitch Tracker* откроется сразу раздел **streams**, а не главная страница
